# Pink
Final project of 'Adaptive websites and automation' course by [HTML Academy](https://htmlacademy.ru/).

## Project features
* BEM
* Adaptive layout
* Sass/SCSS
* Zeplin
* Pixel Perfect
* Gulp
* Some JS (mouse-events, popup)
